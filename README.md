# Rath Yatra 2026 - Premium Interactive Web Experience

A high-fidelity, interactive Rath Yatra celebration website built with HTML, CSS, and Vanilla JavaScript. 
Features dynamic physics, premium animations, a canvas-based particle engine, and an elegant glassmorphism UI.

## Features
- **Cinematic Canvas Effects**: Golden dust, falling flowers, floating orbs, and light flares dynamically responding to user interaction.
- **Rope Pull Physics Engine**: Click/Touch and drag to physically pull the chariot with realistic tension and release mechanics.
- **Progressive Milestones**: Evolving environment (sky glow, particle count) based on how far the chariot is pulled.
- **Victory Celebration**: Reaching 100% triggers an explosion of particles and a premium greeting card display.
- **Glassmorphism UI**: Beautiful, clean UI overlay for quotes and progress tracking.

## Usage
1. Open `index.html` in your browser.
2. Drag the "PULL" handle left to pull the Rath.
3. Keep pulling until 100% progress is reached.
